# TimeUnitDisplay2D

Displays one specific time unit, those being hours, minutes and seconds.

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/TimeUnitDisplay2D.git
```

## Documentation
https://github.com/ChillCube/TimeUnitDisplay2D/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon TimeUnitDisplay2D
```

## For Developers
```bash
push-all-addons
```

## Dependencies

- [NumberDisplay](https://github.com/ChillCube/NumberDisplay)
- [Clock](https://github.com/ChillCube/Clock)
